## 🌟 Rate Standards

#### Rate
- Longer than 10 seconds
- Verified Legitimately
- No secret way or hacks
- Playable Gameplay
- Simple Decoration
#### Featured
- All requirements for a rated level
- Decent Gameplay
- Good Decoration
#### Epic, Legendary  
- All requirements for a featured, epic level
- Fun Gameplay
- Better Decoration
#### Mythic  
- All requirements for a featured, epic, legendary level
- Best Gameplay
- Best Decoration
- Original Level
